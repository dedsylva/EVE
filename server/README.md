# EVA - Backend 

## Features
APIS for receiving/delivering requests from frontend. Currently we have two kinds:

* Node APIS
  - Weights
  - Blood Pressure
  - Heart Monitor
  - Scheduling Exams/Appointments

* Python APIS
  - Skin Cancer Exam
  - Skin Diseases Exam
